Detailed Description

Base class for a spectrum buffer. Spectrum buffers are responsible for transforming the spectrum using FFT and then managing the data created. The buffer superclass can implement the FFT using what ever method they want and generate what ever type of data they want from the spectrum, is the displacements, slope, etc.
