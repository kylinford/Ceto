Detailed Description

Custom container that allows items to be looked up with a key like a dictionary while also retaining what order the items where added to the container like a queue all with O(1) complexity. Implemented by using a dictionary to store linked list nodes and a linked list to store dictionary keyValuePairs.

Modification of the DictionaryQueue where the value is also the key.
