Detailed Description

The WaveSpectrum component is responsible for creating the wave displacement, slope and foam textures. A spectrum of frequencies is created based on the wind speed, wind direction and wave age. This spectrum is then transformed from the frequency domain to the spatial domain using a Fast Fourier Transform algorithm. The wave displacements can be transformed on the CPU using threading or on the GPU. The slope and jacobians (foam) are always performed on the GPU.
