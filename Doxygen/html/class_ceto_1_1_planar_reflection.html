Detailed Description

Class to provide reflections using the planar reflection method.

Note - since this method presumes that the wave plane is flat having rough wave conditions can cause reflection artefacts. You can use the reflection roughness to blur out the reflections to hide these artefacts. You wont have sharp reflections but thats the best that can be done with this method.
