Detailed Description

Calculates the projection VP matrix and interpolation matrix. The projector VP matrix is used by the overlay system to project the overlays onto the ocean mesh. The interpolation matrix is used to convert a screen space mesh position into the world position on the projection plane.

NOTE - The projection math is carried out in double precision as it was found the math is quite sensitive to precision errors.
